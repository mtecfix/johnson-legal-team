### 3. AVVO

**Cost:** FREE (basic profile)
**Time to setup:** 30-45 minutes
**Expected leads:** 3-8/month (Q&A answers drive significant traffic)

#### Step-by-Step:

1. **Go to** https://www.avvo.com/for-lawyers/avvo-profile
2. **Search your name** — Avvo creates profiles for ALL licensed attorneys automatically from bar records
3. **Click "Is this you? Claim and update your profile for free"**
4. **Create login** with firm email
5. **Verify identity** (usually via bar number confirmation)
6. **Complete ALL profile sections** (each section adds to your Avvo rating):

   **Profile Completion Checklist (each item increases your rating):**
   - [ ] Professional headshot uploaded
   - [ ] Practice areas selected (all that apply)
   - [ ] Detailed biography written (500+ words)
   - [ ] Office address and contact info
   - [ ] Bar admissions and year admitted
   - [ ] Education (law school with graduation year)
   - [ ] Work experience (all positions with dates)
   - [ ] Legal awards / recognitions
   - [ ] Speaking engagements
   - [ ] Publications / articles
   - [ ] Professional associations (State Bar MI, OCBA, etc.)
   - [ ] Pro bono / volunteer work
   - [ ] Languages
   - [ ] Fee structure (free consultation, payment plans, etc.)

7. **Request peer endorsements** — ask 3-5 colleagues to endorse you on Avvo
8. **Request client reviews** — ask 3-5 past clients to leave reviews
9. **START ANSWERING Q&A** (this is the biggest lead driver):
   - Go to https://www.avvo.com/free-legal-advice
   - Filter by your practice areas + Michigan
   - Answer 3-5 questions per week
   - Be helpful, specific, and include "feel free to contact our office" at the end
   - Each answer links back to your profile
   - High-quality answers get upvoted = more visibility

#### Avvo Rating Scale (aim for 8.0+):
- 1.0-3.9: Attention (no activity)
- 4.0-5.9: Concerned (incomplete profile)
- 6.0-6.9: Good (basic completion)
- 7.0-7.9: Very Good (complete profile + some endorsements)
- 8.0-8.9: Excellent (full profile + endorsements + reviews)
- 9.0-10.0: Superb (full profile + many endorsements + many reviews + Q&A activity)

### 5. STATE BAR OF MICHIGAN LAWYER REFERRAL SERVICE (LRS)

**Cost:** $150/year (base)
**Time to setup:** 30-60 minutes (approval takes 1-2 weeks)
**Expected leads:** 5-10/month (HIGH intent, pre-qualified)

#### Step-by-Step:

1. **Confirm eligibility:**
   - [ ] Active attorney in good standing with State Bar of Michigan
   - [ ] Maintain malpractice insurance (required)
   - [ ] Have an enhanced profile in SBM online Member Directory

2. **Set up enhanced SBM Member Directory profile:**
   - Go to https://www.michbar.org
   - Log in to member area
   - Navigate to Member Directory settings
   - Complete enhanced profile (bio, practice areas, photo, contact info)

3. **Apply for LRS Panel:**
   - Go to http://www.michbar.org/programs/lawyerreferral_panel
   - Submit application online or call Panel Coordinator Monique Smith at (517) 346-6323
   - Email: msmith@michbar.org

4. **Select practice areas (4 included in base fee):**
   - Criminal Law / Expungements
   - Family Law / Domestic Relations
   - Probate / Estate Planning
   - Personal Injury
   - (Additional areas: $25 each)

5. **Select judicial circuits (2 included in base fee):**
   - 3rd Circuit Court (Wayne County)
   - 6th Circuit Court (Oakland County)
   - (Additional circuits: $25 each, or $300 for ALL Michigan circuits)

6. **Pay $150 registration fee** (Visa or MasterCard)

7. **How referrals work once approved:**
   - Consumers call (800) 968-0738 (8:30 AM - 4:45 PM, M-F)
   - Trained referral assistants assess the caller's needs
   - Caller pays $25 for a 25-minute consultation
   - LRS matches caller to a panel attorney by practice area + location
   - YOU receive the referral by email/phone
   - You provide the 25-minute consultation
   - If client hires you — you set your own fees from there

#### Why This Is High-Value:
- These are people ACTIVELY LOOKING for a lawyer RIGHT NOW
- They've already paid $25 — invested, serious
- Matched by practice area + geography = highly relevant
- State Bar reputation = trust
- $150/year = one converted case pays for 10+ years of membership

## FREE TOOLS AND FRAMEWORKS

### 1. n8n — Workflow Automation Engine (100% Free Self-Hosted)

**What it is:** Open-source workflow automation (like Zapier but free and self-hosted)
**GitHub:** https://github.com/n8n-io/n8n (45k+ stars)
**License:** Sustainable Use License (free for self-hosted, commercial use OK)
**Cost:** $0 self-hosted on the existing AWS account

**Why n8n for JLT Lead Processing:**
- Visual workflow builder (no-code)
- 660+ pre-built integrations
- Gmail trigger node (polls inbox — can replace or supplement inbox-monitor)
- Webhook nodes (receive from any form/API)
- HTTP request nodes (call jude-leads Lambda API)
- AI nodes (connect to Bedrock/Claude for lead qualification)
- Google Sheets integration (simple lead tracking backup)
- Cron scheduling (run workflows every X minutes)
- Deduplication logic built-in
- Self-hosted = no data leaves your AWS account

**Deployment Plan:**
- Run on a t3.micro EC2 instance ($0 free tier or ~$8/month after)
- Or run in a Docker container on existing infrastructure
- Uses PostgreSQL or SQLite for state (can use existing RDS or local)
- Expose via internal endpoint only (not public)

**Key Workflows to Build:**
1. **Gmail → Lead Intake:** Poll Gmail every 15 min → parse → deduplicate → POST to jude-leads API
2. **Form Submissions → CRM:** Webhook catches form data → enrich → store
3. **Lead Notification (when enabled):** New lead → format → SMS to Attorney Johnson
4. **Follow-Up Automation (when enabled):** Lead not contacted in X hours → auto-email
5. **Daily Digest:** Aggregate all leads → morning summary email

### 2. Twenty CRM — Open-Source Salesforce Alternative (100% Free Self-Hosted)

**What it is:** Modern CRM with pipeline management, contacts, tasks, email integration
**GitHub:** https://github.com/twentyhq/twenty (45k+ stars)
**License:** AGPL-3.0 (free, self-hosted, commercial use OK)
**Cost:** $0 self-hosted

**Why Twenty for JLT:**
- Modern UI (looks like Notion meets Salesforce)
- Lead pipeline with drag-and-drop stages
- Contact management
- Activity timeline (all interactions with a lead in one view)
- Custom fields (case type, score, urgency, source)
- API access (jude-leads can push data in)
- Email integration
- Task management
- Self-hosted = client data stays on your servers (important for legal)

**Deployment Plan:**
- Run on t3.small EC2 ($0 free tier or ~$17/month)
- Docker Compose deployment
- PostgreSQL database (can share existing RDS)
- Access via private URL (staff only)

**Alternative (Simpler):** If Twenty is overkill for now, use **Google Sheets** as the CRM (free, Attorney Johnson already has Gmail). n8n can push leads directly to a shared Google Sheet with columns for all lead data + status column for manual review.
