Google OAuth Configuration

Step 1: Create Google OAuth Credentials

  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable "Google+ API" or "Google Identity Services"
  4. Go to "Credentials" → "Create Credentials" → "OAuth 2.0 Client ID"
  5. Application type: "Web application"
  6. Add Authorized JavaScript origins:
    • http://localhost
    • https://yourdomain.com
  7. Add Authorized redirect URIs:
    • http://localhost/johnson-legal-team/api/google-callback.php
    • https://yourdomain.com/api/google-callback.php
  8. Copy your Client ID and Client Secret

Step 2: Update config.php

Add these lines to your config.php:

define('GOOGLE_CLIENT_ID', 'your-client-id.apps.googleusercontent.com');
define('GOOGLE_CLIENT_SECRET', 'your-client-secret');
define('GOOGLE_REDIRECT_URI', 'http://localhost/johnson-legal-team/api/google-callback.php');

Step 3: Test

Click "Sign in with Google" on the login page
